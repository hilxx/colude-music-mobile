```
yarn start  // 启动项目
yarn test   // 测试项目
yarn build  // 打包项目
```
---
React 全家桶 (包含 hooks) 以及 immutable 数据流为基础打造的一款高质量的移动端音乐类 WebApp 。

### 后端部分:
采用 github 上开源的 NodeJS 版 api 接口 NeteaseCloudMusicApi，地址 https://github.com/hilxx/NeteaseCloudMusicApi
